# 页面设计

## 首页 DashboardView

- 路径：`/`
- 统计卡片：员工总数、部门总数、用户总数（仅 admin 可见）
- 快捷操作：新增员工、新增部门、员工列表、部门列表、用户管理（仅 admin）

## 登录页 LoginView

- 路径：`/login`
- 字段：用户名、密码
- 登录成功后跳转首页，token 存 localStorage，自动获取用户信息
- 未登录访问其他页面自动跳转登录
- 被禁用用户登录会提示"user is disabled"

## 员工管理

### 员工列表 EmployeeList

- 路径：`/employees`
- 多选表格，支持批量删除
- 搜索：关键字搜索（姓名/邮箱/电话/身份证）
- 排序：可选字段（ID/姓名/邮箱/状态/创建时间）+ 升降序
- 分页：支持切换每页条数
- 表格：多选框、ID、姓名、邮箱、电话、部门、状态、创建人、更新人、操作
- 操作：编辑、删除

### 员工表单 EmployeeForm

- 路径：`/employees/new`、`/employees/:id/edit`
- 字段：姓名、邮箱、电话、身份证号、部门（下拉）、状态（单选）
- created_by/updated_by 由后端从 JWT 自动填充

## 部门管理

### 部门列表 DepartmentList

- 路径：`/departments`
- 多选表格，支持批量删除
- 搜索：关键字搜索（部门名称）
- 排序：可选字段（ID/名称）+ 升降序
- 分页：支持切换每页条数
- 表格：多选框、ID、名称、描述、创建人、更新人、操作
- 操作：编辑、删除

### 部门表单 DepartmentForm

- 路径：`/departments/new`、`/departments/:id/edit`
- 字段：名称（必填）、描述

## 用户管理（仅 admin）

### 用户列表 UserList

- 路径：`/users`
- 多选表格，支持批量禁用
- 搜索：用户名/角色
- 表格：多选框、ID、用户名、角色、状态、创建时间、操作
- 操作：分配角色（弹窗多选）、启用/禁用切换

## 布局

经典后台管理系统布局：
- 顶部导航栏：左侧 logo + 菜单（首页/员工管理/部门管理/用户管理），右侧用户名 + 角色标签 + 退出
- 用户管理菜单仅 admin 可见
- 下方主内容区：浅灰背景，router-view
- 表单页顶部：el-page-header 返回 + 标题
- 列表页分区：上方搜索/筛选栏（el-card），下方表格 + 分页（el-card）
- Element Plus 中文 locale
